== RestaurantQueue

WIP Rails app to organize restaurant where you've been and new ones you want to try. Never have to deal with the dreaded "IDK, where do _you_ want to go?" question again.

* Ruby 2.2.4
* Rails 4.2.10
* Postgres 0.21

Hosted on heroku at: https://restaurantqueue.herokuapp.com/
